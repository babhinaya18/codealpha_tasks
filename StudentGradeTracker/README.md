\# 🎓 Student Grade Tracker



A modern Java Swing-based Student Grade Tracker application designed to manage student records, calculate grades automatically, analyze academic performance, and generate detailed reports. The application features a professional dashboard interface with Light/Dark themes, real-time statistics, search functionality, and persistent data storage using file handling.



\---



\## 🚀 Features



\### 👨‍🎓 Student Management



\* Add new student records

\* Update existing student information

\* Delete student records

\* Auto-generated Student IDs

\* Duplicate entry validation

\* Easy-to-use form interface



\### 📊 Grade Calculation



\* Automatic grade generation based on marks

\* Grade categories:



&#x20; \* A (90–100)

&#x20; \* B (80–89)

&#x20; \* C (70–79)

&#x20; \* D (60–69)

&#x20; \* F (Below 60)



\### 📈 Performance Analytics



\* Average marks calculation

\* Highest marks tracking

\* Lowest marks tracking

\* Class topper identification

\* Total student count

\* Real-time dashboard updates



\### 📋 Grade Distribution Analysis



\* Grade A distribution

\* Grade B distribution

\* Grade C distribution

\* Grade D distribution

\* Grade F distribution

\* Percentage-based statistics



\### 🔍 Search \& Filtering



\* Search students by ID

\* Search students by Name

\* Dynamic table filtering

\* Instant record lookup



\### 💾 Data Persistence



\* Save student records to file

\* Load saved records automatically

\* Generate detailed reports

\* File-based storage system



\### 🎨 Modern User Interface



\* Professional dashboard layout

\* Light Theme

\* Dark Theme

\* Color-coded grade display

\* Progress bar score visualization

\* Responsive table design
