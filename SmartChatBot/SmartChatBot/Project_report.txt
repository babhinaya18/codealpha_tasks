PROJECT OBJECTIVE

The objective of SmartChatBot is to provide students with an intelligent desktop assistant capable of helping with study planning, placement preparation, internship guidance, resume analysis, and academic support. The system uses JavaFX for the graphical user interface and MySQL for persistent data storage.

---

FEATURES

• Self-Learning Chatbot
• Runtime Memory System
• Intent Detection
• Sentiment Analysis
• Study Plan Generator
• Resume Analyzer
• Placement Guidance
• Internship Guidance
